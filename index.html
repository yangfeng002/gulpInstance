    不支持 file:// 协议

    看起来，你正在使用 file:// 协议访问 mgui 文档首页。因为 mgui 文档内容需要进行异步加载，你需要使用 http:// 协议或 https:// 协议来访问本文档。

    如果下载了完整的 mgui 源码包，且系统上已经安装有 Node.js 环境，只需按照如下步骤即可立即正常访问文档：

    1. 进入 mgui 源码目录；
    2. 执行 npm install；
    3. 执行 npm start；
    4. 在浏览器中访问 http://127.0.0.1:8088。

    你也可以访问在线文档：
